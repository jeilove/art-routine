## [v0.5.0] - 2026-03-28 (Final Stability Patch)
### Fixed (비로그인 데이터 유실 완결 해결)
- **로직 정합성**: 하이드레이션(로컬 데이터 로밍) 대기 로직 도입으로 데이터 유실 원천 차단.
- **타임존 오차 수정**: UTC 기준에서 Local(현지 시간) 기준으로 날짜 계산 방식을 통일하여 한국 시간 오차 현상 해결.
- **유령 데이터 강제 정화**: 기존 28개 미만 조각 계정에 대해 샘플 데이터 30개를 강제 재주입하여 '빈 화면' 현상 종결.
- **상태 동기화**: 대시보드 진입 시 주기(Cycle)가 실시간으로 최신 데이터에 맞춰 갱신되도록 수정.

### Added
- **AppInit 전역 컴포넌트**: 앱 최상단에서 데이터 무결성을 실시간으로 감시 및 복구.
- **Glassmorphism 로딩 화면**: 데이터 로딩 중 미려한 전용 로더를 표시하여 사용자 경험 향상.
- **SSP(Strict Sequential Protocol)**: 에이전트의 '행동 후 보고' 원칙을 명문화한 글로벌 지침 강화.

## [v0.4.x] - 2026-03-28
### Added
- **비로그인(Guest) 자동 시딩**: 신규 사용자 접속 시 30일치 샘플 데이터를 자동으로 생성하여 핵심 기능 즉시 체감 유도.
- **버전 로커(VersionLogger)**: 브라우저 콘솔(F12)을 통해 현재 배포된 앱의 정확한 버전을 실시간 확인 가능.

### Improvements
- **배포 프로세스 강화**: 사용자 보고 전 GitHub Push 성공 여부 선제 확인 절차 도입.
